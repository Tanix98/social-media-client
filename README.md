# Workflow Course Assignment

### Built with:

* Prettier
* ESLint
* Lint-staged
* Jest
* Cypress
* Husky
* Babel

## Project setup

1. Clone the workflow branch:

```bash
git clone -b workflow --single-branch https://github.com/Tanix98/social-media-client
```

2. Install dependencies

```bash
npm install
```

3. Build project

```bash
npm run build
```
